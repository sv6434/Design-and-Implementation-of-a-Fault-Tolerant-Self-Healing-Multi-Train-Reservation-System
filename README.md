This repositary has the complete source code for the Design and Implementation of a Fault Tolerant Self-Healing Multi Train Reservation System having features like:
1. Core booking logic for two independent trains
 Non-overlapping seat allocation
 Dynamic fare calculation
 Fault Detection Unit (FDU)
 Predictive Monitor for repeated faults
Self-Healing Controller with:
Seat Correction Mode
Fare Compensation Mode
Full Recovery Mode

 Event-level logging using a configurable Healer Log module
 Multiple Verilog testbenches for validation
